
 1. To install this module into ONEPIECE Framework, run the following command:
    `git submodule add https://github.com/onepiece-framework/op-module-model.git asset/module/model`
 1. After installation, please perform the following steps:
    `php Init.php`
    When you run init.php, the following functions are enabled:
         1. GitHooks is enabled and only valid branch names that have passed CI/CD can be pushed.
